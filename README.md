# 🚀 Task Management Web App

A full-stack role-based task management system where users can create projects, assign tasks, and track progress efficiently. Built using the MERN stack with secure authentication and real-time task tracking features.

---

## 🔗 Live Links

- 🌐 Frontend: https://assignment-task-orcin.vercel.app  
- ⚙️ Backend: https://assignment-task-ckag.onrender.com  

---

## 📌 Features

- 🔐 User Authentication (Signup / Login) using JWT
- 👥 Role-Based Access Control (Admin / Member)
- 📁 Project Management (Create & manage projects)
- ✅ Task Management (Create, assign, update status)
- 📊 Dashboard with:
  - Total tasks
  - Completed tasks
  - Pending tasks
  - Overdue tasks
- 🔄 Task status tracking (TODO / IN_PROGRESS / DONE)

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Axios
- React Router DOM
- Tailwind CSS

### Backend
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication

---

## 🧠 System Design

- RESTful APIs for all operations
- Secure authentication using JWT tokens
- Role-based authorization middleware
- Proper MongoDB schema relationships between:
  - Users
  - Projects
  - Tasks
